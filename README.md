## WHAT I’M BUILDING

**`Offline Music Player`** — AB test audio file versions without a DAW or a subscription. Built for artists and engineers who need a fast answer, not a full session.

**`Local LLM File Assistant`** — Personal AI wired into my own machine to surface files and automate the boring parts of my workflow. Less searching, more building.

**`Conversational Lead Intake`** — For clients who ghost quote forms. A step-logic chat interface that feels like a conversation and saves structured lead data to Prisma on the backend.

**`Anti-Suite Native Tools`** — Why pay for all of Figma when you only need the gradient editor? Lightweight native tools that carve out the one feature worth keeping — built exact, no bloat, owned outright.
